# 🍃 QuranFlow - Modern Quranic Experience

**QuranFlow** is a full-stack, performance-optimized web application designed to provide a seamless reading and learning experience of the Holy Quran. Built with **Next.js 16** and **Node.js**, it features a clean UI, fast navigation using SSG, and a customizable reading environment.

## 🚀 Live Demo
- **Frontend:** [https://quranflow-client.vercel.app/]*
- **API Server:** [https://quranflow-server.vercel.app](https://quranflow-server.vercel.app)

## ✨ Key Features
- **Fast Static Generation (SSG):** Optimized page loading for Surahs using Next.js static generation.
- **Dynamic Content:** Fetches real-time data from a custom-built Node.js/Express API.
- **Interactive Sidebar:** Easy navigation through Surahs and Juz.
- **Customization:** Dedicated settings panel for adjusting font sizes and reading modes.
- **Mobile Responsive:** Fully optimized for all screen sizes with a custom App-like mobile drawer.
- **Animated Experience:** Smooth pre-loaders and transitions for a premium feel.

## 🛠️ Tech Stack
### Frontend
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel

### Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Middleware:** CORS, JSON Parser
- **Deployment:** Vercel (Serverless Functions)
